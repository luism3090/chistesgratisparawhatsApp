--------------------------------------------- comentarios de los campos: -----------------------------------------


cant_show_interstitial --> cantidad de veces para que apareza el interstitial por defecto tiene 5

------

band_show_boton_crear_chiste -->  	0 = no hace ninguna condicion (omite la funcionalidad)
									1 = si - Oculta el boton

---------


band_new_version_app --> esta bandera mostrara el tipo de mensaje al usuarios para informarle de que ya hay una nueva actualizacion que puede ser un mensaje Toast o un mensaje en forma de modal

0 = No : no hace la condicion para mandar los mensajes  (omite la funcionalidad)

1 = mensaje por medio de toast

2 = muestra el mensaje por medio de una modal con opciones cancelar y actualizar 



------

new_version_app -->  colocar la nueva version de la app (2.2 o 2.3 o 2.4 o 2.5) ya subida a google play y compararla con la que tiene el usuario en la aplicacion esto ya esta en el codigo (bajo una condicion) esta condicion se verá efectuada siempre  cuando este activa alguna de las banderas que le corresponden 


------

mjs_new_version_app --> Mensaje a salir si hay una nueva version de la app (Al cumplirse una condicion)

Mensaje =  'Hay una nueva versión de chistes gratis para whatsApp descargala ya desde google play'

------

band_old_version_app --> Bandera para condicionar que ya actualicen la aplicacion si la version del usuario ya esta muy viejita  

0 = no --> No hace la funcionalidad de comparar la version mas vieja de la app con la version que tiene el usuario

1 = si  :

(CASO EXTREMO ya que si tienen una version mu viejita los obligara actualizar afuerza) 
 
  muestra el mensaje en una modal con la unica opcion de actualizar y ya no deja entrar a la app (caso extremo)



------

old_version_app  -->  esta es la version que queremos que si el usuario la tiene instalada en su celular 
                            al ser una version ya muy vieja queremos que la atualice 

 de momento la version mas vieja es la 2.0

------

mjs_old_version_app --> mensaje de cuando ya debe actualizar la app afuerza porque ya esta muy vieja esa version

mensaje = 'Debes actualizar la aplicación desde la google play para poder seguirla usando'

-------

total_chistes_por_dia --> cantidad de chistes por dia que se dejaran publicar por defecto tiene 2


-------

band_crear_chistes_revision -->  verificar si los chsites pasan a formulario de revision antes de ser publicados o no y se publican en automatico

1 = Si ser verfican

0 = no se verifican
